# Environmental News & Awareness Portal

A comprehensive web platform for environmental news, climate dashboard, and community-driven issue reporting.

## Features
- **User Authentication**: Secure signup and login.
- **Climate Dashboard**: Visualizations and data on environmental changes.
- **News Portal**: Real-time updates on global and local environmental news.
- **Issue Reporting**: Users can report environmental issues in their community.
- **Responsive Design**: Works perfectly on mobile and desktop.

## Tech Stack
- **Backend**: Flask (Python)
- **Database**: SQLite (SQLAlchemy)
- **Frontend**: HTML5, Vanilla CSS, JS
- **Deployment**: Prepared for Render

## How to Run Locally
1. Install dependencies: `pip install -r requirements.txt`
2. Run the application: `python app.py`
3. Visit `http://127.0.0.1:5000`

## Deployment
This project is configured for deployment on Render using the included `Procfile` and `requirements.txt`.
